# 大明朝官网 - 分阶段开发计划

## 开发策略

**核心理念**: "界面先行，功能后续" - 先构建完整的视觉体验，快速上线获得用户反馈，再逐步完善功能

**优势分析**:
- ✅ 快速上线，占领市场先机
- ✅ 提前获得用户反馈，验证产品方向
- ✅ 分阶段投入，降低风险
- ✅ 持续更新，保持用户关注度

## 第二阶段：核心功能接入 (3-4周)
> **目标**: 实现用户系统和护照申请功能，让用户能够真正"入籍大明"

### 2.1 数据库设计和部署 (3-4天)
**数据库设计**:
```sql
-- 用户基础信息表
CREATE TABLE users (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  username TEXT UNIQUE NOT NULL,
  email TEXT UNIQUE NOT NULL,
  password_hash TEXT NOT NULL,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 大明身份信息表
CREATE TABLE ming_profiles (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  user_id INTEGER REFERENCES users(id),
  chinese_name TEXT NOT NULL,        -- 中文姓名
  courtesy_name TEXT,                -- 字号
  birthplace TEXT NOT NULL,          -- 籍贯
  social_class TEXT NOT NULL,        -- 出身：农/工/商/读书人
  official_rank INTEGER DEFAULT 1,   -- 官职等级
  wealth INTEGER DEFAULT 100,        -- 银两
  prestige INTEGER DEFAULT 0,        -- 声望
  avatar_url TEXT,                   -- 头像
  seal_image TEXT,                   -- 个人印章
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 成就系统表
CREATE TABLE achievements (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  user_id INTEGER REFERENCES users(id),
  achievement_type TEXT NOT NULL,
  achievement_name TEXT NOT NULL,
  description TEXT,
  earned_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

**技术任务**:
- Cloudflare D1数据库创建
- Drizzle ORM配置和连接
- 数据库迁移脚本编写
- 基础CRUD API开发

### 2.2 用户认证系统 (4-5天)

- 邮箱注册/登录

**界面开发**:
- 古风登录/注册页面

**技术重点**:

## 资源投入规划

### 开发资源
- **第一阶段**: 1名全栈开发者 (你自己)
- **第二阶段**: 考虑增加1名前端开发者
- **第三阶段**: 考虑增加后端开发者和设计师
- **第四阶段**: 组建3-5人的开发团队

### 预算规划
- **基础设施**: Cloudflare免费层起步，预计月费用$10-50
- **域名费用**: 约$10-20/年
- **第三方服务**: 图片存储、邮件服务等约$20-100/月
- **营销推广**: 第三阶段后考虑投入$500-2000/月

### 风险控制
- **技术风险**: 使用成熟技术栈，降低技术风险
- **市场风险**: 分阶段投入，及时调整产品方向
- **资金风险**: 控制成本，依靠免费服务起步
- **时间风险**: 设置合理的里程碑，允许适度延期
