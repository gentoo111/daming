# 大明朝官网 - 项目文档

## 项目简介

大明朝官网是一个多元宇宙人生体验平台，以大明朝为起点的沉浸式历史文化体验系统。用户通过申请"大明护照"开始，在不同的历史场景中体验各种人生选择，探索传统文化与现代科技的融合。

## 快速开始

### 环境要求
- Node.js 18+
- npm 9+

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

项目将在 http://localhost:3000 启动

## 核心理念

**"古代形式，现代功能"** - 用传统文化的外在形式，承载现代用户的真实需求：

- 🏛️ **护照申请** - 用户注册的古风包装
- 👥 **朝堂议事** - 社区讨论的雅致呈现  
- 📜 **科举考试** - 知识竞赛的文化载体
- 💰 **商贸投资** - 虚拟股市的古代版本
- 🕯️ **赛博烧纸** - 数字祭祀的创新功能
- 🌳 **古代树洞** - 情感宣泄的匿名平台

## 项目愿景

打造全球最大的中华文化数字体验平台，让海内外华人在数字世界中体验和传承中华文明，在现代科技与传统文化的融合中找到文化认同和归属感。
